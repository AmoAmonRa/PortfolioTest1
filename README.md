# Modern Portfolio Website

A responsive, modern portfolio website built with HTML, CSS, and JavaScript. This portfolio showcases projects, skills, and experience in a clean, visually appealing design with dark/light theme support.

## Features

- Responsive design that works on all device sizes
- Dark/light theme toggle with localStorage persistence
- Smooth scrolling and animated navigation
- Interactive elements with hover effects and animations
- Project showcase with tilt effect
- Skill visualization with animated progress bars
- Contact form with validation
- Scroll progress indicator
- Back to top button
- Performance optimized with lazy loading and efficient animations

## Project Structure

```
portfolio/
├── index.html          # Main HTML structure
├── styles.css          # All CSS styles
├── script.js           # All JavaScript functionality
├── README.md           # This documentation file
```

## Technologies Used

- **HTML5**: Semantic markup for structure and accessibility
- **CSS3**: Modern styling with Flexbox, Grid, and CSS variables
- **JavaScript**: Interactive functionality with modern ES6+ features
- **Responsive Design**: Mobile-first approach with media queries
- **Web Animations**: CSS transitions and JavaScript animations

## How to Use

1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. The portfolio is ready to use - no build process required

## Customization

To customize this portfolio:

1. **Content**: Edit `index.html` to update text content
2. **Styling**: Modify `styles.css` to change colors, fonts, and layout
3. **Functionality**: Update `script.js` to modify interactive elements

### Theme Customization

The portfolio supports both dark and light themes. The theme preference is saved in localStorage and respects system preferences by default.

### Contact Form

The contact form uses the `mailto:` protocol to open the user's default email client. To change this behavior, modify the `script.js` file in the contact form handler section.
